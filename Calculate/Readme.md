# Graphical Calculator

A simple and interactive graphical calculator built with the Flet framework in Python. This calculator supports basic arithmetic operations as well as advanced mathematical functions.

## Features

- Basic operations: addition, subtraction, multiplication, and division.
- Advanced functions: square, absolute value, π, square root, sine, cosine, tangent, and cotangent.
- Clear and delete functionalities.
- Responsive and user-friendly UI.

## Requirements

- Python 3.x
- Flet library

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/saeed-94/graphical-calculator.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Calculate
   ```
3. Install the required dependencies:
   ```bash
   pip install flet
   ```

## Libraries

| Library | Version  |
|---------|----------|
| Python  | 3.x      |
| Flet    | Latest   |

## Operations

| Operation          | Symbol | Description                    |
|--------------------|--------|--------------------------------|
| Addition           | +      | Adds two numbers               |
| Subtraction        | -      | Subtracts one number from another|
| Multiplication     | *      | Multiplies two numbers         |
| Division           | /      | Divides one number by another  |
| Square             | x²     | Squares a number               |
| Absolute value     | abs    | Returns the absolute value     |
| Pi                 | π      | Inserts the value of π         |
| Square root        | √x     | Calculates the square root     |
| Sine               | Sin()  | Calculates the sine            |
| Cosine             | Cos()  | Calculates the cosine          |
| Tangent            | Tan()  | Calculates the tangent         |
| Cotangent          | Cot()  | Calculates the cotangent       |
| Clear              | C      | Clears the input               |
| Delete character   | <      | Deletes the last character     |

## Usage

Run the calculator application:
```bash
python main.py
```
